## 🧪 Lab Writeup Format

Each lab follows a structured methodology:

### 📝 Lab Overview

* Platform: HTB / CyberDefenders
* Difficulty: Easy / Medium / Hard
* Category: DFIR / Threat Hunting / Malware Analysis

### 🔍 Scenario

Brief description of the incident or challenge.

### 🛠️ Tools Used

* Wireshark
* Splunk
* Autopsy
* Volatility

### 🔎 Investigation Steps

2. Log analysis
3. Artifact extraction
4. Correlation

* Attacker behavior
* Techniques used

What was learned from the lab.
